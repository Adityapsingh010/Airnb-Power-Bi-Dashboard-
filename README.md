# Airbnb Power BI Dashboard

This project presents an interactive Airbnb dashboard built using Power BI to analyze and visualize Airbnb data. The dashboard includes insights on pricing trends, room types, availability, and neighborhood patterns, helping users explore market trends and understand rental performance effectively.

## Key Features
- Analysis of listing prices by neighborhood.
- Breakdown of room types and availability trends.
- Interactive charts and KPIs to showcase key metrics.
- Geographical visualization of Airbnb data.

## How To Use
1. Download or clone this repository.
2. Open the Power BI file and connect it to the data source.
3. Explore the reports and visuals in the dashboard.

## About
The purpose of this project is to demonstrate the power of data visualization and insights extraction using Airbnb data. It is an example of transforming raw data into meaningful insights for decision-making.
